Code should be beautiful. APIs should feel like magic. Technology should serve humans, not the other way around.

My journey through technology has taught me that the most impactful innovations don't just work—they feel right. From the first line of Requests to the architectural decisions in Certifi, every choice reflects a core belief: developers are humans first, and our tools should honor that humanity.

The Work That Matters

Each project represents years of thoughtful iteration, community feedback, and an obsession with developer experience.

Requests: HTTP for Humans™

The library that redefined what HTTP could feel like in Python. Simple, elegant, powerful. "There are 9 million reasons to use Requests," and developers worldwide have discovered them. ~25M daily downloads · The gold standard for HTTP libraries

Certifi: Trust Database for Humans™

The invisible guardian of internet security. A carefully curated collection of Root Certificates that validates the trust between your code and the rest of the world. ~20M daily downloads · Security infrastructure you can depend on

Pipenv: Python Packaging for Humans™

Bringing sanity to Python dependency management. The official PyPA-recommended tool that finally made Python packaging feel natural and intuitive. Official PyPA recommendation · Dependency management, simplified

The Hitchhiker's Guide to Python

An opinionated guide to Python development that's become essential reading for developers worldwide. Best practices, project structure, and the Python philosophy, distilled. Community treasure · Thousands of contributors · Essential reading

Records: SQL for Humans™

Database queries without the cognitive overhead. Raw SQL power with Python elegance—because sometimes you just want to write SQL and get results back.

Maya: Datetimes for Humans™

Time handling that doesn't make you want to scream. Intuitive datetime manipulation that finally makes working with time zones and intervals feel natural.

Design Philosophy

Every line of code I write is guided by a simple principle: technology should amplify human potential, not complicate it.

Simplicity is the ultimate sophistication. Often attributed to Leonardo da Vinci, though the exact origin is disputed. The sentiment perfectly captures the essence of elegant design.

This isn't just about clean APIs or elegant syntax. It's about recognizing that behind every import requests is a human being trying to solve a problem, build something meaningful, or maybe just get home to their family a little earlier.

When millions of developers trust your code to power their applications, you carry a responsibility that goes beyond functionality. You're shaping how people think about problems, how they approach solutions, and ultimately, how they feel about their craft. This philosophy extends beyond code to documentation, community building, and the entire developer experience ecosystem.
